Sincerely yours,

This website is an fun experiment, during which I applied the HTML, CSS and Javascript skills I picked up, and to use this site as a portfolio to show potential employers.

Besides the basic HTML/CSS to create the basic layout I also implemented some Javascript to connect the gallery overlay to the pictures and the corresponding side text.

Responsive breakpoints have been implemented, to allow the website to be viewed on mobile devices such as tablets and smart phones.

Tic Tac Toe

Tic Tac Toe was a game app for which I did the concept and design. Starting from sketches, I worked my way up to conceptual wireframes and finally did colors, UI elements and styles in the Adobe programs.

Besides the classical tictactoe the app is supposed to offer an ancient greek game mode called "Terni Lapili", a 9x9 Meta tic tac toe and a quattro level, which are playable against the computer as well as a friend on the same device. A competitive online-mode was intended for a later update.
